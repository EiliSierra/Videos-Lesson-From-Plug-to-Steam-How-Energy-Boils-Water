## PRODUCTION NOTES

### Key Visual Elements to Create:

1. **Energy flow animation** (Part 2)
   - Electrical particles flowing through wire
   - Glow effects
   - Color changes (blue/yellow for electricity)

2. **Molecule animation** (Part 2)
   - Water molecules as circles/spheres
   - Movement speed variations
   - Vibration effects
   - Spacing changes

3. **Component creation** (Part 4)
   - Gray circle "pop" animation when component appears
   - Bounce effect
   - Selection highlight
   - Drag trail (optional)

4. **Color change animation** (Step 5)
   - Gray to Orange transition
   - Glow/pulse effect
   - Ripple animation

5. **UI feedback**
   - Text selection (blue highlight)
   - Cursor states (hover, click, drag)
   - Snap-to-grid effects
   - Confirmation animations

#### VISUAL EMPHASIS POINTS:
- Slow motion on key movements (click and drag)
- Visual effects when components appear (gray circles)
- Cursor highlight for easy following
- Keyboard indicators when Shift is mentioned (visual "SHIFT" indicator)
- Soft zoom on important areas
- Color change to orange for external component

#### ON-SCREEN TEXT OVERLAYS:
- "Click anywhere on the graph panel"
- "Double-click to rename"
- "Shift + Click = External"
- Step numbers: "Step 1 of 5", etc.
- Component names as they're created
