# From Plug to Steam: How Energy Boils Water - Video Scripts

Educational video scripts for 8th Grade Physical Science (PS3) using Cell Collective modeling platform.

## üìö Project Overview

This repository contains professional video scripts and production materials for the educational lesson **"From Plug to Steam: How Energy Boils Water"**. The lesson teaches students about energy systems, energy transfer, and system modeling through hands-on activities using the Cell Collective platform.

### üéØ Learning Objectives

Students will be able to:
- Identify key components in a simple energy system
- Show the direction of energy flow between system parts
- Set up simulations that model energy-driven changes
- Test what happens when energy flow is disrupted
- Use simulation results to explain how energy input leads to observable results

### üéì Grade Level & Standards

- **Grade:** 8th Grade
- **Subject:** Physical Science
- **Standard:** PS3 (Energy)
- **Platform:** Cell Collective
- **Modeling Approach:** Systems thinking and computational modeling

## üé¨ Production Notes

### Key Features of These Scripts

‚úÖ **Student-Centered Narrative** - Conversational tone that resonates with 8th graders
‚úÖ **Detailed Timing** - Precise timestamps for each section
‚úÖ **Visual Specifications** - Complete descriptions of animations and effects needed
‚úÖ **Technical Corrections** - Addresses common platform misconceptions:
   - "Click anywhere on the graph panel" (NOT "click + button")
   - "Click click and drag" - emphasis on dragging action
   - "Gray circle" appears (NOT "box")
   - "Shift + Click = Orange = External" - clear visual feedback

### Animation Requirements

1. **Energy Flow Animations**
   - Electrical particles flowing through wires
   - Glow effects and color transitions
   - Blue/yellow electricity visualization

2. **Molecule Animations**
   - Water molecules as animated spheres
   - Speed variations showing heat transfer
   - Vibration and spacing changes

3. **UI Feedback Animations**
   - Component "pop" animations when created
   - Gray to orange color transitions
   - Cursor highlighting and movement effects

### Technical Specifications

**Video:**
- Resolution: 1920x1080 (1080p minimum)
- Frame rate: 30fps or 60fps
- Format: MP4 (H.264)
- Aspect ratio: 16:9

**Audio:**
- Voice: Clear, friendly American English
- Music: Background at 20-30% volume
- Sound effects: Subtle (clicks, pops, confirmations)
- Mix: Voice 100%, Music 20-30%, SFX 40-50%
